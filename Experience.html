ประสบการณ์

ผมมีโอกาสได้ฝึกงานที่บริษัท ซีพีเอฟ (ประเทศไทย) จำกัด (มหาชน) ซึ่งเป็นประสบการณ์ที่มีค่าอย่างยิ่ง ผมได้เพิ่มพูนทักษะในการเขียน HTML, CSS และ JavaScript เพื่อนำไปใช้ในการพัฒนาเว็บไซต์ นอกจากนี้ยังได้เรียนรู้การใช้งาน Arduino เพื่อพัฒนาระบบเซ็นเซอร์ที่สามารถตรวจจับความร้อน ความชื้น และอุณหภูมิภายในโรงงาน ซึ่งช่วยเสริมสร้างความรู้และทักษะในการทำงานที่เกี่ยวข้องกับเทคโนโลยีในสถานการณ์จริงอย่างมีประสิทธิภาพ

โครงการที่อบรม

โครงการอบรม การเขียนโปรแกรมภาษา Python เบื้องต้น รุ่นที่ 4

โครงการอบรม การพัฒนา Mobile Web Application ด้วย HTML, CSS, JavaScript, Bootstrap และ php รุ่นที่ 5

โครงการวิศวะไฟฟ้าสร้างสรรค์เสริมปัญญา : การตรวจจับและรู้จำใบหน้า โดยใช้ Python ร่วมกับ OpenCV และ Tensorflow รุ่นที่ 1
