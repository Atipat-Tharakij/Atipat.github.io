โปรเจคและโครงงานที่เคยทำ

​ออกแบบ​เว็บไซต์​ข้อมูลแบบเรียล​ไทม์

เขียน code พัฒนา​เว็บไซต์​ข้อมูลแบบเรียล​ไทม์​โดยจะมีการเขียน Html เพื่่อออกแบบโครงสร้างและกำหนดองค์ประกอบ​ต่างๆของหน้าเว็บ เขียน Css ในการจัดรูปแบบและการแสดงผลขององค์ประกอบ​โดย กำหนดสี ขนาด ฟอนต์ การจัดวาง ให้หน้าเว็บสวยงาม เขียน Javascrip​ เพิ่มฟังก์ชันการทำงาน เช่น การตอบโต้กับผู้ใช้ การโหลดข้อมูลแบบเรียลไทม์ การคลิกปุ่ม และการปรับเปลี่ยนเนื้อหาของ HTML และ CSS โดยไม่ต้องโหลดหน้าเว็บใหม่ ในส่วนของผลการรัน:

​การเขียนโค้ดโปรแกรมสั่งอาหาร

สร้างโปรแกรมสั่งอาหารโดยใช้ภาษา Java ซึ่งประกอบด้วยการพัฒนาไฟล์หลักๆ ได้แก่ 1.Customer.java: จัดการการสั่งอาหารของลูกค้า 2.CustomerQueue.java: จัดการคิวของลูกค้า 3.Order.java: จัดการรายการอาหาร การทำงานของโปรแกรมจะให้ลูกค้าเลือกเมนูอาหารโดยการพิมพ์หมายเลข 1 ถึง 5 ซึ่งแต่ละหมายเลขจะเชื่อมโยงกับรายการอาหารที่มีให้เลือก ลูกค้าสามารถเลือกอาหารได้โดยการพิมพ์หมายเลข 1 ถึง 10 เพื่อระบุรายการที่ต้องการจากนั้นลูกค้าจะต้องกรอกชื่อเพื่อทำการสั่งอาหารและจัดเก็บข้อมูลในคิวในส่วนของผลการรัน:

สร้างวงจรสัญญาณเตือนไฟไหม้โดยใช้โปรแกรม Proteus 7

สร้างวงจรสัญญาณเตือนไฟไหม้ ทำงานโดยการตรวจจับก๊าซหรือควันที่เกิดจากไฟไหม้ รวมถึงการตรวจจับเปลวไฟโดยตรง เมื่อมีการตรวจพบไฟไหม้จะมีการส่งสัญญาณเตือนผ่าน LED ที่ควบคุมโดย Arduino ทำให้เป็นระบบเตือนภัยไฟไหม้แบบอัตโนมัติ

วงจรปลดล็อครหัสผ่าน

วงจรนี้เป็นระบบรักษาความปลอดภัยสำหรับประตูที่ใช้รหัสผ่าน โดยมีการทำงานแบบอินเตอร์แอคทีฟระหว่างผู้ใช้กับไมโครคอนโทรลเลอร์ การแสดงผลของระบบจะใช้ไฟ LED สีต่าง ๆ เพื่อแจ้งสถานะและ 7-Segment Display เพื่อแสดงข้อมูล ทำให้ระบบนี้มีความปลอดภัยและสะดวกในการใช้งาน

ปรับแต่งเสียงให้เหมาะสมกับการฟังโดยใช้ Filter Bank

ใช้ MATLAB เพื่อออกแบบตัวกรอง Filter Bank โดยการเลือกและตั้งค่าตัวกรองที่แบ่งช่วงความถี่ของเสียง เช่น การใช้ฟังก์ชัน designfilt และ fir1 เพื่อสร้างตัวกรองที่มีความถี่ตัดที่ต้องการ กรองสัญญาณเสียงโดยนำสัญญาณเสียงผ่านตัวกรองที่ออกแบบไว้โดยใช้ฟังก์ชัน filter เพื่อแยกและปรับแต่งเสียงในแต่ละช่วงความถี่ จากนั้นรวมสัญญาณที่ผ่านการกรองกลับเป็นสัญญาณเสียงที่ปรับแต่งแล้วใช้ฟังก์ชัน freqz และเครื่องมือวิเคราะห์อื่นๆ ใน MATLAB เพื่อตรวจสอบและปรับแต่งคุณสมบัติของตัวกรองเพื่อให้ได้เสียงที่มีคุณภาพตามต้องการ
